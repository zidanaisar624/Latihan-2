# Program Kriptografi

Program ini mengimplementasikan tiga sandi: sandi Vigenere, Playfair, dan Hill dengan antarmuka pengguna grafis (GUI).

## Cara Menjalankan Program:
1. Buka menggunakan VS Code
2. Klik run untuk menjalankan program (bisa pilih dengan debugging ataupun tidak)
3. Tulis random pada Enter Key
4. Tulis random pada Enter Plain Text
5. Untuk mengenkripsi tinggal klik Encrypt
6. Untuk mengdeskripsi bisa copy-paste hasil Encrypt ke Enter Plain Text dan klik Decrypt
